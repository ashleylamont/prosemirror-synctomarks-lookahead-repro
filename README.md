# Prosemirror syncToMarks lookahead issue reproduction

This repository contains a minimal reproduction of an issue with Prosemirror's `syncToMarks` function, specifically related to lookahead behavior.

You can view the example at [ashleylamont.github.io/prosemirror-synctomarks-lookahead-issue/](https://ashleylamont.github.io/prosemirror-synctomarks-lookahead-repro/).

Alternatively, you can have a look at the code directly in this repository. [main.ts](./src/main.ts) contains the relevant code for the Prosemirror editor setup and bug repro.
